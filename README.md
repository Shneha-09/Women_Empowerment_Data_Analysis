# Women_Empowerment_Data_Analysis

This project presents a data analysis dashboard created to celebrate International Women's Day. The dashboard analyzes key indicators of women's empowerment across multiple countries using data visualization.

## Project Objective

The goal of this project is to explore and visualize important factors related to women's empowerment such as education, workforce participation, political representation, and gender equality trends.

## Dataset Information

The dataset includes the following fields:

- Country
- Year
- Female Literacy Rate (%)
- Female Labor Force Participation (%)
- Women in Parliament (%)
- Gender Gap Index

The dataset contains data from multiple countries across several years to analyze trends and patterns in gender equality.

## Dashboard Insights

The dashboard provides insights into:

- Average female literacy rate across countries
- Women workforce participation levels
- Representation of women in parliament
- Gender equality trends over time
- Comparison of countries based on women empowerment indicators

## Tools Used

- Microsoft Power BI
- Data Visualization
- Data Analysis

## Dashboard Features

- Interactive visualizations
- Country-wise comparisons
- Trend analysis by year
- KPI cards highlighting key metrics
